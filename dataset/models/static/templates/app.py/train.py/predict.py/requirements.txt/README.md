🌿 Plant Disease Detection Using Deep Learning

📖 Project Overview

Plant Disease Detection Using Deep Learning is a final-year Computer Science Engineering project that uses Artificial Intelligence and Computer Vision to identify diseases in plant leaves from images. The application enables users to upload a leaf image and receive an instant prediction of the disease along with treatment and prevention recommendations.

The project aims to support farmers, agricultural researchers, and students by providing a fast, accurate, and user-friendly disease detection system.

---

🎯 Problem Statement

Plant diseases significantly reduce crop yield and quality. Traditional disease identification requires expert knowledge and manual inspection, which can be time-consuming, expensive, and unavailable in many rural areas.

This project addresses the problem by developing an AI-powered system capable of detecting plant diseases from leaf images, allowing for early diagnosis and timely treatment.

🔄 Project Workflow

1. Collect the PlantVillage dataset.
2. Preprocess the images.
3. Train a Deep Learning model.
4. Save the trained model.
5. Upload a leaf image through the web application.
6. Predict the disease.
7. Display treatment and prevention recommendations.

📈 Expected Outcome

The application should accurately identify plant diseases from leaf images and provide useful recommendations, helping users take timely action to reduce crop losses.
